== Description ==

<a href='https://ultimatecaptcha.com/'>Ultimate Captcha</a> helps you to protect your website by controlling fake accounts, spam comments. This plugin prevents brute force logins on your WordPress website, also you can avoid fake accounts created by robots.

The plugin uses reCAPTCHA and it's totally free. You can protect the following pages:

<ul>
<li>WordPress Login Page</li>
<li>WordPress Registration Page</li>
<li>WordPress Password Reset Page</li>
<li>WooCommerce Login Page</li>
<li>WooCommerce Registration Page</li>
<li>WooCommerce Password Reset Page</li>
<li>Post Comments</li>

</ul>

<h4>PRO FEATURES</h4>

<strong>Please note:</strong> Using Ultimate Captcha is free, reCAPTCHA is a Google product, we're not charging for the integration with reCAPTCHA. The following features are improvements and they're optional.

<ul>
<li><a href='https://ultimatecaptcha.com/extensions/password-strength/'>Password Strength</a> - This add-on helps you to protect your users' accounts by managing and monitoring the strength of their passwords.</li>
<li><a href='https://ultimatecaptcha.com/extensions/aweber/'>AWeber</a> - This will help you to start with your e-mail marketing campaign. It will add a checkbox to the registration form and ask users to register for your newsletter/e-mail list.</li>
<li><a href='https://ultimatecaptcha.com/extensions/custom-pages/'>Custom Registration & Login Pages</a> - This allows you to create custom registration, login and password reset pages. You can also override the default WP pages.</li>
</ul>

<h4>CUSTOM REGISTRATION AND LOGIN PAGES</h4>
<ul>
<li><a href='https://ultimatecaptcha.com/testing/registration-page/'>WordPress Custom Registration Page</a> - You can take additional information from your users, even you can ask them to input a password.</li>
<li><a href='https://ultimatecaptcha.com/testing/login-page/'>WordPress Custom Login Page</a> - This is a custom login page which can be used to override the default WordPress login page.</li>
<li><a href='https://ultimatecaptcha.com/testing/password-recover/'>Custom Password Reset Page</a> - You can also override the default WP reset link.</li>
</ul>


== Installation ==

1. Upload `ultimate-captcha` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
